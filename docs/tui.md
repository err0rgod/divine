# TUI guide

The Textual TUI is an operations console, not a chat interface. Start it with `divine` or
`divine tui`.

Pages cover:

- Dashboard
- Provider configuration and credentials
- Models and aliases
- Routing policies and fallback chains
- CLI-agent profiles
- Server settings
- Usage statistics and provider health
- Diagnostics and redacted logs

Use Tab and arrow keys for navigation, `d` to return to Dashboard, and `q` to quit. The layout
prioritizes stable keyboard operation and readability in both light and dark terminals.

Credentials screens show references and availability, not values. Content logging remains off
unless explicitly enabled in configuration with an understanding of the privacy impact.
